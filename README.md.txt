# DULEME OS

## Présentation

DULEME OS est le système d'information de DULEME AND CIE.

Il centralise l'ensemble des activités du cabinet :

- Site internet
- Relationship OS
- Gestion des missions
- Dossier DULEME
- Bibliothèque DULEME
- Gestion des contenus
- Assistant DULEME
- Espace Client
- Automatisations

L'objectif est de fournir un environnement unique, simple, évolutif et indépendant des outils utilisés.

# Philosophie

DULEME OS est développé selon les principes suivants :

- simplicité avant complexité ;
- architecture modulaire ;
- une donnée saisie une seule fois ;
- données exportables ;
- IA assistante, humain décisionnaire ;
- indépendance vis-à-vis des fournisseurs.

---

# Développement

Toute évolution doit respecter :

- le Manuel Fondamental ;
- le PRD ;
- le Data Dictionary ;
- le Guide de Développement.

En cas de contradiction, le Manuel Fondamental prévaut.
